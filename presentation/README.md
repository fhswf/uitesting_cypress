# UI-Testing mit Cypress

## Was ist Cypress?

- Framework für das Testing von Web-Anwendungen
- Open-Source
- Ähnlich wie Selenium
- Meiner Meinung nach deutlich besser, einfacher und flexibler

## Installation

```
npm install cypress
```

![Cypress Beispiele](images/examples.PNG)

```
cypress open
```
